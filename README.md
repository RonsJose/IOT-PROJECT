# IOT-PROJECT

This is the Smart System integration for automobiles. It aims to improve old and new cars that do not have current and up to date safety features, by implementing them at a much lower cost. It meets a lot of the sustainable development goals that are set by the UN, such as good health and well-beaing, industry innovation an infrastructure, sustainable cities and communties & responsible consumption and production.

Currently it features:
A website that shows all the information on a dashboard, 
Distance sensor with audio cues to indicate distance,
Alcohol sensor that detects high alcohol and locks the doors and alerts a chosen email of the location and intoxicated state,
A temperature and humidity sensor for the drivers comfort,
A GPS sensor to get accurate location information,
A camera that can be used when reversing into a parking bay or any other activities,
A heart rate & blood oxygen monitor that report your current health status,
A rfid locking system
